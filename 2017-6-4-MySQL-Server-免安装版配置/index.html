#####MySQL 安装

服务注册

管理员启动cmd,进入bin目录

mysqld -install

MySQL初始化

会自动生成data目录，无root密码

mysqld –initialize-insecure

MySQL启动

net start mysql

密码修改

输入

mysql -u root -p
Enter password: 此处空格键即可

输入 及密码修改为root

set password = password(‘root’);

MySQL NET HELPMSG 3534 错误处理

删除 data目录，重新使用 mysqld –initialize-insecure 命令进行初始化即可
